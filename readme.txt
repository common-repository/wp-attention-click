== Description ==
**WP Attention Click** is a plugin that is focused on *capture the attention of the visitor of your page* before they leave him.

The average number of visits at a global level is one 8 seconds, then this visit closes your website or another website goes. That the plugin does is for that visit to take one more action on your website before leaving.

------------

For this **WP Attention Click**  uses 2 methods currently:

- #### Bar
It shows in the upper part of the page a bar where you can call the attention of the visitor, with the personalized button you can have them click in order to have an attention. A very used case is to increase social networks like Facebook, Instagram, etc...

- #### Popup
A picture appears on the content, this box makes the visitor read quickly what it is about to be able to read and click on the attention button. A widely used case has been used to promote a book or sell something.

Both options have very similar but separate characteristics to be able to have different objectives, as both the Bar and the Popup have their options separately.


**BAR** It has the following characteristics:
>**Activete Bar**
>
>**Direction:** Choose the address where the bar appears
>
> #### Bar customizer - *Options to customize the bar to get attention*
>
>- Message you want to refer: You can add a small photo, > spaces, colors, and fonts.
>- Background color
>- Text color
>- Padding up and/down
>
> #### Button Custom (Optional): *If you want to show the custom action button you must put this code in the editor `{{button}}`*
>- Background color
>- Text color
>- Text of the button
>
> #### Action onClick: *Event configuration click to get attention*
>- **Time to appear:** Time it takes for the bar to appear (in seconds)
>- **Show again:** Time it returns to appear (in days)
>- **Text clickeable:** *Active* The text of the bar will become a link. *Inactive* The text of the bar is not clickable but you can use the button as a clickable item.
>- **Open link in a new page:** Open the link on a new page, if this option is not activated then open it on the same page.
>- **Action:** *Url* By clicking on the action bar or button this redirects to a customizable URL here. *Whatsapp* Redirection to the WhatsApp application
>- **Url**
>
> #### Advanced:  *Advanced customization options*
> - **Show in country:** Select the countries in which you want to display If the option is "All" or empty, then it will be shown in all countries.


**Popup** It has the following characteristics:
>**Activete Popup**
>
> #### Popup customizer: *Options to customize the popup to > get attention*
>
> - **Content of the popup:** You can add a small photo, spaces, colors, and fonts.
> - **Background color**
> - **Text color**
> - **Padding**
>
> #### Button Custom (Optional): *If you want to show the > custom action button you must put this code in the editor `{{button}}`*
> - **Background color**
> - **Text color**
> - **Text of the button**
>
> #### Action onClick: *Event configuration click to get attention*
> - **Time to appear:** Time it takes for the popup to appear (in seconds)
> - **Show again:** Time it returns to appear (in days)
> - **Text clickeable:** *Active* The text of the bar will become a link.  *Inactive* The text of the bar is not clickable, but you can use the button as a clickable item.
> - **Open link in a new page:** Open the link on a new page, if this option is not activated then open it on the same page.
> - **Url**
>
> #### Advanced: *Advanced customization options*
> - **Show in country:** Select the countries in which you want to display If the option is "All" or empty, then it will be shown in all countries

== Installation ==
This section describes how to install the plugin and get it working.

1. Unzip the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
